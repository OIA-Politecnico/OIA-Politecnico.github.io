## A - Feed the Beast

Notar que el predicado `P(x) = es posible hacerlo por x dias` cumple las
condiciones para hacer busqueda binaria. Entonces hacemos busqueda binaria sobre
la respuesta para encontrar la maxima cantidad de dias que es posible completar.

```c++
#include <bits/stdc++.h>
using namespace std;

#define forr(i,a,b) for(int i = int(a); i < int(b); ++i)
#define forn(i,n) forr(i,0,n)
#define all(x) begin(x),end(x)

using ll = long long;

// division redondeada hacia arriba
__int128 cd(__int128 a, __int128 b) { return (a+b-1) / b; }

int main() {
    int n, b, x;
    cin >> n >> b >> x;
    
    vector<int> a(n);
    forn(i, n) cin >> a[i];
    
    ll daily = accumulate(all(a), 0ll);
    ll lo = 0;                     // sin duda es posible 0 dias
    ll hi = ll(b) * x / daily + 1; // sin duda es imposible. Tambien funcionaria poner 1000000000001 (13 digitos)
    
    while (hi - lo > 1) {
        ll mi = (lo + hi) / 2;
        
        __int128 need = 0;
        forn(i, n) need += cd(a[i] * mi, x);
        if (need <= b) lo = mi;
        else hi = mi;
    }
    
    cout << lo << "\n";
}
```

> En esta solucion se usa `__int128`, pero es posible evitarlo si tenemos un
> poco mas de cuidado con las cuentas.

## B - Phoenix and Puzzle

Podemos usar 2 triangulos para armar un cuadrado de 1x1. Si queremos hacer un
cuadrado de NxN, usamos N^2 cuadrados, o 2xN^2 triangulos.

Como muestra la figura del enunciado, tambien podemos usar 4 triangulos para
hacer un super cuadrado de 1x1. Si queremos hacer uno de NxN, usamos 4xN^2
triangulos.

Como la maxima cantidad de triangulos no es tan alta, podemos construir todas
las cantidades validas, meterlas en un arreglo ordenado, y despues usar
`std::binary_search()` para chequear si una cantidad dada en el input aparece en
ese arreglo.

```c++
#include <bits/stdc++.h>
using namespace std;

#define forr(i,a,b) for(int i = int(a); i < int(b); ++i)
#define forn(i,n) forr(i,0,n)
#define pb push_back
#define all(x) begin(x),end(x)

int main() {
    
    vector<int> a;
    forr(i, 1, 23000) {
        a.pb(i*i*2);
        a.pb(i*i*4);
    }
    sort(all(a));
    
    int n; cin >> n;
    
    forn(_, n) {
        int k; cin >> k;
        if (binary_search(all(a), k)) cout << "YES\n";
        else cout << "NO\n";
    }
}
```

		std::cout << "0" << std::endl;

			cout << P[i] << " ";
		};
		return 0;
	};

	sort(P.begin(), P.end());
	vector<ll> R(n);
	ll hi = n - 1;
	for (ll i = 0; i < n; i += 2) {
		R[i] = P[hi];
		hi--;
	};

	ll cheap = 0;
	for (ll i = 1; i < n; i += 2) {
		R[i] = P[hi];
		hi--;

		if (i + 1!= n) {
			cheap++;
		};
	};

	cout << cheap << endl;
	FORR(i, 0, n) {
		cout << R[i] << " ";
	};
	return 0;
};

## F - Santa Claus and Keyboard Check

Hacemos una primera pasada registrando los pares de teclas intercambiadas. En
particular lo anotamos en una tabla que asocia cada letra con la que esta
intercambiada.

Es posible que la misma tecla aparezca varias veces intercambiada con distintas
teclas. Para detectar esto, hacemos otra pasada por las cadenas viendo si en
alguna posicion hay un intercambio distinto al registrado en la tabla.

> Se pueden hacer las dos cosas en una sola pasada pero me parecio mas facil
> codearlo asi

Despues iteramos por todas las letras y nos guardamos un par de caracteres por
cada letra que no esta asociada con si misma. Aparte, para no registrar cada par
dos veces (una por cada letra que lo compone), solo guardamos el par al pasar
por la letra menor de las dos (o sea, solo si la letra registrada en la tabla es
mayor)

```c++
#include <bits/stdc++.h>
using namespace std;

#define forr(i,a,b) for(int i = int(a); i < int(b); ++i)
#define forn(i,n) forr(i,0,n)
#define pb push_back
#define all(x) begin(x),end(x)
#define sz(x) (int((x).size()))

int main() {
    string s, t;
    cin >> s >> t;
    
    int n = sz(s);
    
    char r[256];
    fill(all(r), 0);
    
    forn(i, n) {
        r[s[i]] = t[i];
        r[t[i]] = s[i];
    }

    forn(i, n) {
        if (r[s[i]] != t[i] || r[t[i]] != s[i]) {
            cout << "-1\n";
            return 0;
        }
    }
    
    vector<pair<char, char>> ps;
    forn(i, 256) {
        if (i < r[i]) {
            ps.pb({(char)i, r[i]});
        }
    }
    
    cout << sz(ps) << "\n";
    for (auto [x, y] : ps) {
        cout << x << " " << y << "\n";
    }
}
```
